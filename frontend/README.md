# Frontend Development Guide

## Структура проекта
- `static/css/` - CSS файлы
- `static/js/` - JavaScript файлы  
- `static/images/` - Изображения
- `templates/` - HTML шаблоны

## Ограничения для фронтенда
Фронтенд разработчик работает ТОЛЬКО в этой папке и не имеет доступа к:
- Бэкенд логике (`api/`, `core/`, `data/`)
- Базам данных
- Конфигурационным файлам сервера

## API Endpoints для фронтенда
Фронтенд общается с бэкендом через API:
- `GET /api/products` - получить список продуктов
- `POST /api/products` - создать продукт
- и т.д.

## Правила разработки
1. Все стили в `static/css/`
2. Весь JavaScript в `static/js/`
3. Шаблоны в `templates/`
4. Не трогать файлы вне папки `frontend/`

## Запуск разработки
Фронтенд работает через Flask - автоматически обновляется при изменениях.
