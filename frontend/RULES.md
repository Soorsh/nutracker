# ОГРАНИЧЕНИЯ ДЛЯ ФРОНТЕНД-РАЗРАБОТЧИКА

## 🚫 ЗАПРЕЩЕНО (изменять)
- `core/` - бизнес-логика
- `data/` - базы данных
- `api/` - API endpoints (только использовать)
- `app.py` - основной сервер
- `.git/` - настройки Git

## ✅ РАЗРЕШЕНО (работать)
- `frontend/` - ВСЯ фронтенд часть
  - `templates/` - HTML шаблоны
  - `static/css/` - стили
  - `static/js/` - JavaScript
  - `static/images/` - картинки

## 📋 ЗАДАЧИ ДЛЯ ФРОНТЕНДА
1. Верстка интерфейса (HTML/CSS) ✓
2. JavaScript логика (AJAX запросы) ✓
3. Адаптивность и UX ✓
4. Валидация форм ✓

## 🔗 API ДЛЯ ОБЩЕНИЯ С БЭКОМ
- GET  `/api/status` - статус сервера
- GET  `/api/products` - список продуктов
- POST `/api/products` - добавить продукт
- И т.д. (будут добавляться по мере разработки)

ПРАВИЛО: Фронтенд НЕ трогает логику, базы и сервер!
